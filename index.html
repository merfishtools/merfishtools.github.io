MERFISHtools

A Bayesian framework for gene expression analysis on MERFISH data.

MERFISHtools implement a Bayesian framework for accurately predicting gene or transcript expression from MERFISH data. On top, differential expression analysis can be performed for two or multiple conditions, including credible intervals for fold change and coefficient of variation, and controlling the expected false discovery rate.

Citation

If you use MERFISHtools, please cite our preprint

Köster, Johannes; Brown, Myles; Liu, X. Shirley (2016): A Bayesian model for single cell transcript expression analysis on MERFISH data. figshare. https://dx.doi.org/10.6084/m9.figshare.3438662.v1

Installation

MERFISHtools can be installed and updated easily via Bioconda.
